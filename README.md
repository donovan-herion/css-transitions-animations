## Setup

To run this project, install it locally using npm:

- Clone this repository
  \$ git clone https://github.com/ewelina-slepko/Pure-CSS-Animation

- Go into the repository
  \$ cd Pure-CSS-Animation

- Install dependencies
  \$ npm install

- Run the app
  \$ npm start
